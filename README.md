# fullcustom-blog
Um projeto de uma página estática completa, compatível e adaptável a diferentes dimensões de tela e dispositivos, concebida utilizando HTML e CSS nativos, com injeções de JavaScript. Focado em design responsivo, estrutura semântica e interatividade com JavaScript.

Estrutura desenvolvida como projeto da disciplina de Interfaces Web, com objetivo de entender desde a prototipação e desenvolvimento até a entrega de um projeto Web com linguagem "crua".
